# 业务与数据流

## 路网编辑 双存储同步
### 添加城市
- Validator 校验输入合法性
- network.add_city 执行
- matrix_graph.add_vertex 邻接矩阵
- list_graph.add_vertex 邻接表
- 两侧同步，任一失败则回滚

### 删除城市
- network.remove_city 执行
- 双结构同步删除顶点
- 级联删除关联道路

### 添加道路
- Validator 校验 from/to/weight
- network.add_road 执行
- 无向图双向添加边
- 有向图单向添加边
- 双结构同步

### 删除道路
- network.remove_road 执行
- 双结构同步删除

### 修改权值
- network.update_road_weight 执行
- 双结构同步更新

---

## 拥堵模拟
### 设置拥堵
- 备份原始权值到 modified_from/to/original_weight 数组
- 调用 simulator.set_congestion 修改矩阵
- 同步更新邻接表权值
- 记录 congestion_active 状态

### 拥堵前后对比
- 拥堵前：运行 Dijkstra 缓存 before_dist / before_prev
- 应用拥堵权值
- 拥堵后：运行 Dijkstra 缓存 after_dist / after_prev
- 逐城市对比距离变化
- 输出对比报告

### 恢复拥堵
- 遍历修改记录数组
- 恢复原始权值到矩阵和邻接表
- simulator.restore_all

### 查看修改记录
- 输出当前拥堵道路 from/to/original_weight/congested_weight

## 文件 IO
### 保存 save_to_file
- 确定路径 参数优先否则用 current_path
- ofstream 打开文件
- 写入文件头：图类型 城市数 道路数
- 遍历城市：写入 ID 名称
- 遍历边：写入起点 终点 权值
- 关闭文件
- 更新 current_file_path

### 加载 load_from_file
- 确定路径
- ifstream 打开文件
- 读取文件头：图类型 城市数 道路数
- 校验：图类型合法性
- 校验：顶点数不超限
- 逐行解析城市 (ID + 名称)
- 逐行解析边 (from + to + weight)
- 校验：城市存在、边不重复、权值合法
- 加载到图结构
- 加载到邻接表和矩阵双结构
- 更新 current_file_path

### 自动加载
- 程序启动时调用
- 优先加载 data/default.txt
- 失败则尝试 data/ 下第一个 .txt

### 切换路网
- system("dir /b") 列出 data/*.txt
- 用户选择
- clear_graph 清空当前图
- load_from_file 加载选中文件

## 结构性能对比
### 内存占用测量
- 调用 get_performance_stats 获取两图内存字节数
- 输出对比

### 遍历速度测量
- 对两图分别运行 DFS + BFS 全程
- std::chrono 计时
- 输出毫秒数对比

### 查找效率测量
- reset_perf_counters 重置计数器
- 遍历全部顶点执行 has_vertex
- 统计比较次数

### 边查询效率测量
- 遍历全部顶点对
- 执行 get_edge_weight
- 统计查询次数

### 综合报告
- 四项指标对比表格
- 输出结论 推荐适用场景
