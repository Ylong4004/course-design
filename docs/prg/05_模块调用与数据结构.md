# 模块调用与数据结构

## 分层架构调用关系
### 入口层
- main.cpp → cli_run 统一入口
- cli_run → MenuSystem 菜单 或 CommandParser 命令行
- 测试模式 → run_all_tests → TestRunner

### UI 层
- MenuSystem → RoadNetwork 路网管理
- MenuSystem → CongestionSimulator 拥堵模拟
- MenuSystem → StructureComparator 性能对比
- MenuSystem → FileManager 文件读写
- MenuSystem → Formatter 格式化输出
- MenuSystem → Validator 输入校验

### CLI 层
- CommandParser → RoadNetwork 路网管理
- CommandParser → CongestionSimulator 拥堵模拟
- CommandParser → StructureComparator 性能对比
- CommandParser → FileManager 文件读写
- CommandParser → 各算法函数 traversal/shortest_path/spanning_tree/topological

### 服务层
- RoadNetwork → GraphBase 指针矩阵
- RoadNetwork → GraphBase 指针邻接表
- CongestionSimulator → GraphBase 操作图
- CongestionSimulator → run_dijkstra 对比分析
- StructureComparator → GraphBase 两套图对比
- FileManager → GraphBase 保存和加载

### 算法层
- traverse_dfs/bfs → GraphBase 参数
- traverse_bfs → Queue 手写队列
- run_dijkstra → GraphBase 参数
- run_floyd → GraphBase 参数
- build_mst_prim → GraphBase 参数
- build_mst_kruskal → UnionFind 并查集
- run_topological_sort → Queue 手写队列
- run_topological_sort → GraphBase 参数

### 存储层
- GraphBase 抽象基类 → 26 个纯虚接口
- AdjMatrix 邻接矩阵 → 实现 GraphBase
- AdjList 邻接表 → 实现 GraphBase

---

## GraphBase 抽象接口 26个纯虚函数
### 存储标识
- get_storage_type → 返回 MATRIX 或 LIST
- get_storage_name → 返回名称字符串

### 图属性查询
- get_vertex_count → 顶点数
- get_edge_count → 边数
- get_graph_type → 有向或无向
- get_max_vertex_count → 最大容量

### 顶点操作
- add_vertex → 添加城市顶点
- remove_vertex → 删除城市及关联边
- has_vertex → 检查存在
- get_vertex → 获取城市信息

### 边操作
- add_edge → 添加道路
- remove_edge → 删除道路
- update_edge_weight → 修改权值
- get_edge_weight → 查询权值
- has_edge → 检查存在

### 遍历支持
- get_neighbors → 获取邻接边列表
- get_all_vertex_ids → 获取全部顶点编号

### 性能统计
- get_performance_stats → 获取统计指标
- reset_perf_counters → 重置计数器

### 调试
- print_graph → 打印图结构

## 核心结构体定义
### City_t 城市顶点
- int id 城市唯一编号
- char name[32] 城市名称

### Edge_t 道路边
- int from 起点城市编号
- int to 终点城市编号
- int weight 权值

### EdgeNode_t 邻接表边结点
- int dest_city 目标城市编号
- int weight 边权值
- EdgeNode_t next 下一邻接边 链表

### AdjListHead_t 邻接表头结点
- City_t city 城市信息
- EdgeNode_t first_edge 第一条邻接边

### MSTResult_t 最小生成树结果
- Edge_t edges 生成树的边集合
- int edge_count 边的数量
- int total_cost 总造价

### PerfStats_t 性能统计指标
- size_t memory_bytes 内存占用字节
- double traverse_time_ms 遍历耗时毫秒
- int find_comparisons 查找比较次数
- int edge_query_count 边查询次数
