# 程序主流程

## 启动阶段
- 程序启动 → 设置 UTF-8 编码 → 调用 cli_run

### 模式分派
- 无参数 → 菜单模式
- --cli → CLI 交互模式
- --gui → Qt GUI 模式
- 命令参数 → 批处理模式

## 菜单模式
### 初始化
- 创建 MenuSystem 菜单系统
- init_network 初始化路网
- load_default_data 加载数据
- 优先加载 default.txt
- 失败则加载第一个可用 txt

### 主菜单循环
- 显示 10 项主菜单
- 用户选择功能编号
- dispatch_choice 分派执行

#### 1 路网编辑
- 新增城市 → 删除城市 → 新增道路
- 删除道路 → 修改权值 → 查看总览 → 查看详情

#### 2 图遍历
- DFS 深度优先遍历
- BFS 广度优先遍历

#### 3 最短路径
- Dijkstra 单源最短路径
- Floyd 多源最短路径

#### 4 最小生成树
- Prim 算法
- Kruskal 算法

#### 5 拓扑排序
- Kahn 算法 + 环路检测

#### 6 拥堵模拟
- 修改权值 → 恢复修改 → 查看记录 → 对比分析

#### 7 结构性能对比
- 内存占用 → 遍历速度 → 查找效率 → 边查询

#### 8 数据文件管理
- 保存 → 加载 → 切换路网 → 设置默认路径

#### 9 帮助
- 功能概览

#### 10 命令行模式
- 切换到 CLI 交互

#### 0 退出系统
- 确认 → 自动保存到当前文件 → 释放资源

## CLI 交互模式
### 命令循环
- 显示提示符 > → 用户输入命令
- tokenize 分词 → dispatch 分派 24 条命令

### 路网编辑
- new_city / del_city / new_road
- del_road / update_road / show_network

### 图算法
- dfs / bfs / dijkstra / floyd
- prim / kruskal / topo

### 拥堵模拟
- congest / restore / congest_list / congest_report

### 文件管理
- save / load / list

### 系统命令
- help / menu / exit / compare

## 批处理模式
- 拼接命令行参数 → execute 执行 → 输出结果 → 退出
