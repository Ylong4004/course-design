# 图算法流程

## DFS 深度优先遍历
### 入口检查
- graph 为空 → ERR_GRAPH_EMPTY
- 起点不存在 → ERR_CITY_NOT_FOUND

### 初始化
- 分配 visited 数组全部 false
- 分配 sequence 序列数组

### 递归访问
- 标记 visited 为 true
- 加入遍历序列
- 获取所有邻接点
- 对未访问邻接点递归调用

### 非连通图处理
- 外层遍历全部顶点
- 未访问的顶点再次启动 DFS

### 返回
- 序列数组和长度

---

## BFS 广度优先遍历
### 入口检查
- graph 为空 → ERR_GRAPH_EMPTY
- 起点不存在 → ERR_CITY_NOT_FOUND

### 初始化
- 分配 visited 数组全部 false
- 创建 Queue 手写队列
- 起点入队并标记 visited

### 队列循环
- 出队顶点 v
- 加入遍历序列
- 获取 v 的邻接点
- 未访问邻接点入队并标记

### 非连通图处理
- 遍历全部顶点
- 未访问顶点入队继续 BFS

### 返回
- 序列数组和长度

---

## Dijkstra 最短路径
### 初始化
- dist 数组全部 INF_WEIGHT
- prev 数组全部 -1
- determined 数组全部 false
- dist[起点] = 0

### 主循环
- 选取 determined 为 false 中 dist 最小的顶点 u
- u 不可达则跳出
- 标记 determined[u] = true
- 获取 u 的邻居
- 松弛：dist[u] + weight < dist[v] 则更新

### 返回
- dist 距离数组
- prev 前驱数组

---

## Floyd 多源最短路径
### 初始化矩阵
- dist[i][j] = 边存在则权值，否则 INF
- dist[i][i] = 0
- next[i][j] = 边存在则 j，否则 -1

### 三重循环
- k = 0 到 V-1
- i = 0 到 V-1
- j = 0 到 V-1
- dist[i][k] + dist[k][j] < dist[i][j] 则更新
- next[i][j] = next[i][k]

### 返回
- dist 距离矩阵
- next 路径矩阵

---

## Prim 最小生成树
### 检查
- 仅适用于无向图
- 图不连通 → ERR_DISCONNECTED

### 初始化
- key 数组全部 INF
- parent 数组全部 -1
- inMST 数组全部 false
- key[0] = 0

### 主循环
- 选取 inMST 为 false 中 key 最小顶点 u
- u 不可达 → 图不连通
- inMST[u] = true
- 遍历 u 的邻居 v
- weight < key[v] 则更新 key 和 parent

### 构建结果
- 根据 parent 生成边集
- 统计总造价

---

## Kruskal 最小生成树
### 检查
- 仅适用于无向图

### 收集边
- 遍历全部顶点获取邻接边
- 无向图只收集 from < to 的边

### 排序
- 按权值升序排列

### 并查集初始化
- 每个顶点自成一个集合
- 路径压缩 + 按秩合并

### 主循环
- 遍历排序后的边
- find(from) ≠ find(to) 则合并
- 加入 MST 边集
- 边数达 V-1 提前结束

### 检查
- MST 边数 ≠ V-1 → 图不连通

---

## 拓扑排序 Kahn 算法
### 检查
- 仅适用于有向图
- 无向图 → ERR_INVALID_INPUT

### 计算入度
- 遍历全部顶点的出边
- 统计每个顶点的入度

### 初始化队列
- 入度为 0 的顶点入队

### 主循环
- 出队顶点加入拓扑序列
- 遍历该顶点邻接点
- 邻接点入度减 1
- 入度变为 0 时入队

### 判断
- 序列长度 == V → 无环
- 序列长度 < V → 存在环
